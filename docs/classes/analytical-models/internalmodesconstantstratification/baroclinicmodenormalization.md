#  BaroclinicModeNormalization

Return the requested analytical normalization factor for baroclinic modes.

> Developer documentation: this item describes internal implementation details.


---

## Declaration
```matlab
 A = BaroclinicModeNormalization(self,norm,k_z,h)
```
## Parameters
+ `self`  InternalModesConstantStratification instance
+ `norm`  normalization convention from `Normalization`
+ `k_z`  vertical wavenumber row vector
+ `h`  equivalent-depth row vector

## Returns
+ `A`  normalization factor row vector
